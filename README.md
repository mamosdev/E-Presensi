# Aplikasi Presensi Geolocation

Starter Project untuk membuat Aplikasi E-Presensi dengan menggunakan laravel 9

## Cara Install

1. install composer

```
composer install
```

2. buat file .env dengan cara berikut

```
cp .env.example .env
```

2. generate public key

```
php artisan key:generate
```

3. jalankan aplikasi

```
php artisan serve
```
